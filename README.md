# Python Calculator

A clean and simple command-line calculator application built with Python. This project demonstrates core Python programming concepts, user input handling, control flow, error handling, and practical software development workflow.

## Features

- Addition, subtraction, multiplication, and division operations
- User-friendly command-line interface
- Floating-point number support
- Division-by-zero error handling
- Modular and readable Python code structure

## Technologies Used

- Python 3
- Git & GitHub
- Visual Studio Code
- Continue Dev AI Coding Assistant
- Qwen 3.5 9B Local LLM

## Development Approach

I developed this project using a local AI-assisted development workflow with:

- **Visual Studio Code** as the development environment
- **Continue Dev** as an AI coding assistant extension
- **Qwen 3.5 9B** running locally as the Large Language Model

The AI tools were used to support software development activities such as code generation, debugging, refactoring ideas, and improving documentation. The project demonstrates my ability to effectively integrate modern AI-assisted tools into a practical programming workflow while maintaining understanding and control of the codebase.
